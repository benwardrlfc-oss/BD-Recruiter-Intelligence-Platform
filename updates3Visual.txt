Core Product Philosophy

This platform is not a data warehouse and not a cluttered financial terminal.

The system should feel like a high-end intelligence assistant for recruiters, designed to deliver:

• market news
• intelligence analysis
• hiring signals
• actionable BD opportunities

The user should receive intelligence → understand it quickly → take action immediately.

The interface must therefore prioritise:

clarity
minimalism
professional authority
ease of action

The experience should feel premium, calm, and intelligent, not overwhelming.

Users should feel like they are using a high-end professional product, similar to the experience of using well-designed modern software like:

Linear
Notion
Arc Browser
Apple Pro apps

The system should never feel like:

a spreadsheet
a CRM
a data terminal
a cluttered dashboard

Design Principles
1. Radical Clarity

Every screen should answer three questions immediately:

What happened
Why it matters
What action should I take

If the interface does not clearly answer these questions, it is too complex.

Information should be layered so that important insights are visible instantly while deeper analysis is available when needed.

2. Minimal Cognitive Load

The platform should avoid:

large dense tables
excessive charts
multiple colours competing for attention
heavy borders

Instead the interface should rely on:

clear cards
clean spacing
structured layout
short summaries

The goal is to allow the recruiter to understand their market in seconds.

3. Calm Professional Aesthetic

The interface should feel:

calm
deliberate
precise

Not:

flashy
busy
noisy

This ensures users trust the platform as a serious intelligence product.

Card Design

Cards should be minimal and spacious.

Style

Rounded corners
Soft shadow
No heavy borders

Example

Cards should feel light and refined.

Interactions
Hover Behaviour

When hovering over cards:

Card lifts slightly.

Example

transform: translateY(-3px)

Shadow deepens slightly.

This gives the interface a responsive feel.

Tab Behaviour

When selecting a tab:

Background subtly changes
Accent border appears

Transitions should be smooth.

Example

transition: all 200ms ease
Animations

Animations should be subtle and refined.

Examples
